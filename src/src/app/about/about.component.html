About

Web3ToolKit is an open-source tool that helps to interact with the web3. It allows to connect to different networks and supports multiple actions such as generating a seed phrase, deriving private keys and public addresses or to check if an address is valid. It mostly works offline. Every action that requires an active internet connection is marked as such.
